# PacmanGame.java
Java Pacman Game with report
This project is a simple console-based Pacman Game developed in Java. The game demonstrates core Object-Oriented Programming (OOP) concepts and basic game logic implementation.

Features:
Grid-based movement system for Pacman
Randomized food placement
Score tracking system
Simple user input controls for movement

This project strongly focuses on Object-Oriented Programming principles such as:
Composition – objects like the game board containing other objects (Pacman, food, etc.)
Aggregation – relationships between game components that can exist independently
Encapsulation
Basic control structures and loops
The main goal of this project is to understand how OOP concepts can be applied in real-world style game development using Java.
